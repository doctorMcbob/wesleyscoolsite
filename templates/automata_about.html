This is a spin off a popular Elementary Cellular Automata


The "rule" is converted to a binary string 256 digits long. Therefore there are 2^256 possible rules. Thats a lot!


To decide if each cell will be alive in the next generation

With the cell being checked, build a binary string in this order

0 1 2

3 @ 4

5 6 7

becomes "01001111"

The string is then converted to an integer, it will be between 0 and 255

We then index our rule with that integer, and if the result is a 1, the cell is alive in the next generation
